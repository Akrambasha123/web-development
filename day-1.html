Document Structure Tags

These tags form the backbone of every HTML document, defining its structure and organization.

Basic Structure Tags

<!DOCTYPE html>: Declares the document type and HTML version (HTML5 in this case).

<html>: The root element that wraps all content on the page.

<head>: Contains meta-information about the document (not displayed on the page).

<meta>: Provides metadata like character set, viewport settings, and more.

<link>: Connects external resources like CSS files to the document.

Image Tag

The <img> tag embeds images in the document. It's a self-closing tag with several important attributes:

  • src: Specifies the image path (required)
  • alt: Provides alternative text for accessibility (required)
  • width/height: Sets display dimensions
  • border: Defines image border (though CSS is preferred)

Text Formatting Tags

HTML provides semantic tags for text formatting to add meaning and structure to your content.

<i>: Italic text - typically used for technical terms, foreign phrases, etc.

<b>: Bold text - for stylistic offset without importance.

<strong>: Strong importance - indicates content with strong importance.

<em>: Emphasized text - adds verbal stress emphasis when read.

<sub>: Subscript - H2O (water molecule).

<sup>: Superscript - 23 = 8 (exponent).

<small>: Small print - for side comments or fine print.

<del>: Deleted text - indicates removed content.

<address>:

Contact: 123 Web St.
- for contact information.

<abbr>: HTML - for abbreviations with title attribute showing full text.

<code>: console.log("Hello"); - for code snippets.

<mark>: Highlighted text - for marked/highlighted content.

<ins>: Inserted text - indicates added content.

<blockquote>:

This is a longer quotation from an external source that appears as a block element.

<q>: Short inline quote - for short quotations.

<cite>: The Art of HTML - for titles of creative works.

Anchor Tag (Links)

The <a> tag creates hyperlinks with several important attributes:

  • href: Specifies the link destination (URL or fragment)
  • target: Controls where the link opens (_blank for new tab)
  • download: Indicates the link should download the resource
  • title: Provides additional information on hover
